Run "MIPS_Orrientation_Sensor - Shortcut" before running the system

To run this program, find out the com port that is connected to the "BlueRadios" connection

Then use this port number in the shortcut. Go into the properties of the shortcut,
and change the number after the path to the com port number. 

Side note: there is a space between the port number and the path
